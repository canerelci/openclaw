# Office Room OpenClaw channel

Bundled OpenClaw channel plugin that joins a Pryva Builder Office Room as a named
room participant.

The channel is disabled by default — it only starts when `channels.office-room` is
enabled in the config, so it stays invisible to deployments that do not run a
Builder suite engine.

## Docs

See `docs/channels/office-room.md` in the OpenClaw repository.
